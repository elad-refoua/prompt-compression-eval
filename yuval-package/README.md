## What this study is

Building MENTI personas the way we do it now takes days. Each persona is a ~33,000-character CPE prompt encoding character, defenses, trigger maps, phase gates, non-linearity rules, and body language. The question driving this study is whether all 33K characters are actually doing work. We compressed Sara's full prompt into a 3,537-character structured persona card, generated 10 conversations from each version using the same 10 therapist inputs (novice, intermediate, skilled, harmful), and set up a blinded expert-rating platform. If trained clinicians cannot reliably tell the two conditions apart, the conclusion is that frontier models already know most of CPE and the long prompt is paying for something cheaper to encode. If they can, the differences tell us *which* CPE elements the model cannot infer on its own -- a map of what prompt length is actually buying.

## What was built since the WhatsApp message (2026-04-07)

- **Stimulus set (n = 20 conversations).** Each conversation was generated by an independent "blind" Claude Code agent with no knowledge of the study design, the other condition, or the hypotheses. Agents ran in parallel; no conversations were excluded; no cherry-picking.
- **Full data-provenance document.** Every generation prompt is archived verbatim. Every methodological choice (including the ones that limit the design) is disclosed in advance, before raters are recruited. The goal is that a reviewer should be able to reproduce every step from this document alone.
- **Live evaluation platform.** Pair-randomized display, forced-choice discrimination, dimensional ratings, free-text reasoning. Data flows to a Google Sheets backend with an admin dashboard. GitHub Pages deployment is already live.
- **Rater-qualification document.** Backed by the published literature on psychotherapy competence rating (Kobak, Muse & McManus, Cicchetti, Koo & Li, ICMJE authorship criteria). Defines minimum qualifications, target N, compensation (~1,000 ILS per rater), IRB checklist, and four tiers of acceptable rater pools.
- **Consent form and IRB-ready materials** for expedited review (synthetic data + licensed-professional participants + modest honorarium).
